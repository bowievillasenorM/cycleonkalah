

RUNNING API (install and setup JAVA 8 and MAVEN in your pc)

- go to /cycleonkalah
- open terminal
- mvn clean package
- go to /target
- java -jar boardgames-0.0.1-SNAPSHOT.jar

RUNNING UI (install node js in your pc)

- go to /cycleonkalah/kalah
- open terminal
- npm install
- npm start
- go to browser localhost:3000
